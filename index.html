An editor for Tracery, made with a simple UI.

For a great tutorial on how to use Tracery(and by extension, this editor), please visit this link.
The simple editor below works in the same way that the JSON editor works, with the difference being that all the rules are written in text fields instead of in a JSON object and the big text area works as the starting symbol for the Tracery grammar.

Either by going to the Github repository for the project, or by sending an email to tracitor[dot]contact[at]gmail[dot]com
